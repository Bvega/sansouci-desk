# Sansouci Desk

PHP 8 + MySQL help desk for Sansouci Puerto de Santo Domingo.

## Current modules

- Auth (login/logout)
- Dashboard
- Tickets (CRUD, mass actions)
- Ticket reply panel (responder.php)
- Client portal:
  - portal_cliente.php → create ticket + emails
  - portal_cliente_tickets.php → view & reply tickets
- Email config (config_correo.php + config_email.php)
- Quick reply templates (plantillas.php)
- Service types (tipos_servicio.php)
- Reports (reportes.php)

DB schema: database/schema_sansouci_desk.sql
